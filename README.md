# BlogNode API
## Hostname
* http://localhost:3000/api
## API
### Official
* Users
	* POST/users/login
	* GET/users/
  * GET/users/:id
  * PUT/users/:id
  * DELETE/users/:id
* Categories
	* POST/categories/login
	* GET/categories/
  * GET/categories/:id
  * PUT/categories/:id
  * DELETE/categories/:id
* Posts
	* GET/posts/
  * GET/posts/:id
  * PUT/posts/:id
  * DELETE/posts/:id
